\abstract{In this paper we present a suite of software tools that allows us to
interact with the \textit{Online Encyclopedia of Integer Sequences}; in
particular, (i)~a \textit{crawler} fetches sequences recursively with respect
to their relations, (ii)~a \textit{pretty printer} represents the same data
stored in the archive using two different formats, namely the old console and
modern Jupyter notebooks, (iii)~a \textit{grapher} shows connections among
sequences by using graph structures.}
